# Rapier

Basic kernel for Raspberry Pi Zero (And 2 in the future). Old version.

# Features:
+ Basic MMU support
+ VideoCore:
    + Reading/writing to mailboxes
    + Framebuffer init
    + Drawing to framebuffer
    + Basic font
+ ATAGs parse
+ Misc library:
     + atoi()
     + hexstring()  
+ Memory management:
     + Allocating/deallocating
